# 🏥 CareBot - AI Health Assistant

CareBot is an AI-powered health assistant built with React and FastAPI.

## Features
- 🤖 AI Health Chatbot (powered by Groq AI)
- 🔐 User Authentication (Register/Login)
- 📅 Appointment Booking
- 📋 Medical Records
- 💊 Medication Tracking

## Tech Stack
- **Frontend**: React.js
- **Backend**: FastAPI (Python)
- **Database**: PostgreSQL
- **AI**: Groq AI (Llama 3)
- **Auth**: JWT Tokens

## Setup
1. Clone the repository
2. Set up backend: `pip install -r requirements.txt`
3. Set up frontend: `npm install`
4. Add your API keys to `.env`
5. Run backend: `uvicorn main:app --reload`
6. Run frontend: `npm start`
